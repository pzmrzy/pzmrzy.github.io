    • Valid Palindrome

    Given a string, determine if it is a palindrome, considering only alphanumeric characters and ignoring cases.

    Report:
    First use re to remove all non-alphanumeric characters with re.sub('[^0-9a-zA-Z]', '', s), then convert the string to lowercase.
Use two pointer from begining and end, to check whether the characters at these position are same.
    Code:
      
    class Solution(object):
        def isPalindrome(self, s):
            """
            :type s: str
            :rtype: bool
            """
            if len(s) == 0:
                return True
            s = re.sub('[^0-9a-zA-Z]', '', s)
            s = s.lower()
            n = len(s)
            l, r = 0, n - 1
            while l < r:
                if s[l] != s[r]:
                    return False
                l += 1
                r -= 1
            return True
