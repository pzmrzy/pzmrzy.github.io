    • Missing Ranges

    Given a sorted integer array where the range of elements are in the inclusive range [lower, upper], return its missing ranges.

    Report:
    We maintaince a need variable. for each number n ,
if n < need, continue
if n = need, need + 1
else form a range (need, n - 1) to result, need = n + 1
    Code:
      
    class Solution(object):
        def findMissingRanges(self, nums, lower, upper):
            """
            :type nums: List[int]
            :type lower: int
            :type upper: int
            :rtype: List[str]
            """
            res = []
            need = lower
            for n in nums:
                if n < need:
                    continue
                elif n == need:
                    need += 1
                else:
                    res.append([need, n - 1])
                    need = n + 1
            if need <= upper:
                res.append([need, upper])
            return map(lambda x: str(x[0]) if x[0] == x[1] else "%s->%s" % (x[0], x[1]), res)
