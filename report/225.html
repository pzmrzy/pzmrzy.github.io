    Using queues to implement stack

    Report:
    We have 2 queues, q1 and q1. When do push, append the number into q1, when do pop or top, if q1 have only one element, just return it. Or, we pop all the element from q1 to q2, leave one in q1, this will be the ewsult, then exchange q1 and q2. For top, we should append the last number in q1 to q2 before exchange.
    Code:
      
    class MyStack(object):
        def __init__(self):
            """
            Initialize your data structure here.
            """
            self.q1 = []
            self.q2 = []

        def push(self, x):
            """
            Push element x onto stack.
            :type x: int
            :rtype: void
            """
            self.q1.append(x)

        def pop(self):
            """
            Removes the element on top of the stack and returns that element.
            :rtype: int
            """
            if len(self.q1) == 1:
                return self.q1.pop()
            while len(self.q1) > 1:
                p = self.q1.pop(0)
                self.q2.append(p)
            self.q1, self.q2 = self.q2, self.q1
            res = self.q2[0]
            self.q2 = []
            return res

        def top(self):
            """
            Get the top element.
            :rtype: int
            """
            if len(self.q1) == 1:
                return self.q1[0]
            while len(self.q1) > 1:
                p = self.q1.pop(0)
                self.q2.append(p)
            res = self.q1[0]
            self.q2.append(self.q1.pop(0))
            self.q1, self.q2 = self.q2, self.q1
            return res

        def empty(self):
            """
            Returns whether the stack is empty.
            :rtype: bool
            """
            return len(self.q1) == 0
