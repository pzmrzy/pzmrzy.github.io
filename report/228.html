    Given a sorted integer array without duplicates, return the summary of its ranges

    Report:
    use a point record the start index, if current num + 1 != next num, append the range to result, and update start pointer.
reminder for the a range with only one number
    Code:
      
    class Solution(object):
        def summaryRanges(self, nums):
            """
            :type nums: List[int]
            :rtype: List[str]
            """
            l = len(nums)
            if l == 0:
                return []
            res, start = [], 0
            for i in range(l - 1):
                if nums[i + 1] != nums[i] + 1:
                    res.append(str(nums[start]) if start == i else "%s->%s" % (nums[start], nums[i]))
                    start = i + 1
            res.append(str(nums[start]) if nums[start] == nums[-1] else "%s->%s" % (nums[start], nums[-1]))
            return res
