    • Sliding Window Maximum

    Get the maximium value of a sliding windows in width k

    Report:
    Use a deque to store the position, and keep number correspond to the position in deque decreasing. If a position out of the sliding window, just pop it. For new number, pop all the number less than it, and append it to the right. The max value of a sliding window is the first corrspond number in deque.
    Code:
      
    from collections import deque
    class Solution(object):
        def maxSlidingWindow(self, nums, k):
            """
            :type nums: List[int]
            :type k: int
            :rtype: List[int]
            """
            n = len(nums)
            res = []
            dq = deque()
            for i in range(n):
                while len(dq) > 0 and dq[0] < i - k + 1:
                    dq.popleft()
                while len(dq) > 0 and nums[dq[-1]] < nums[i]:
                    dq.pop()
                dq.append(i)
                res.append(nums[dq[0]])
            return res[k - 1:]
