    • Strobogrammatic Number II

    A strobogrammatic number is a number that looks the same when rotated 180 degrees (looked at upside down).Find all strobogrammatic numbers that are of length = n.

    Report:
    for n = 1, ['0', '1', '8'], for n = 2, ['11', '88', '69', '96', '00']
for n = 3, insert [0, 1, 8] to the middle of solution of n = 2
for n = 4, insert [11, 88, 69, 96, 00] to the middle of solution of n = 2
so we can get the result recursively base on n is odd or even
    Code:
      
    class Solution(object):
        def findStrobogrammatic(self, n):
            """
            :type n: int
            :rtype: List[str]
            """
            n1 = ['0', '1', '8']
            n2 = ['11', '88', '69', '96', '00']
            if n == 1:
                return n1
            if n == 2:
                return n2[:-1]
            mid = (n - 1) / 2
            if n % 2 == 0:
                return [p[:mid] + c + p[mid:] for c in n2 for p in self.findStrobogrammatic(n - 2)]
            else:
                return [p[:mid] + c + p[mid:] for c in n1 for p in self.findStrobogrammatic(n - 1)]
