    Given an array of meeting time intervals consisting of start and end times [[s1,e1],[s2,e2],...] (si < ei), find the minimum number of conference rooms required.

For example,
Given [[0, 30],[5, 10],[15, 20]],
return 2.
    Report:
    keep the meeting end time in a heapq, for a new meeting, if its start time late than the earlest meeting end time, replace it with new end time, else push the new end time into queue
the tot length of the queue is the result
    Code:
      
# Definition for an interval.
# class Interval(object):
#     def __init__(self, s=0, e=0):
#         self.start = s
#         self.end = e
import heapq
class Solution(object):
    def minMeetingRooms(self, intervals):
        """
        :type intervals: List[Interval]
        :rtype: int
        """
        intervals.sort(key = lambda x: x.start)
        q = []
        for i in intervals:
            if q and i.start >= q[0]:
                heapq.heapreplace(q, i.end)
            else:
                heapq.heappush(q, i.end)
        return len(q)
