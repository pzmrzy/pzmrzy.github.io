    Given an array of n integers nums and a target, find the number of index triplets i, j, k with 0 <= i < j < k < n that satisfy the condition nums[i] + nums[j] + nums[k] < target.

    Report:
    Fisrt sort the array. choose a number i, the use two pointer j start from next number and k from end
if the sum less then target, then all the combinition of i, j, j+1 to i, j, k less than target. Then j + 1
else k - 1

    class Solution(object):
        def threeSumSmaller(self, nums, target):
            """
            :type nums: List[int]
            :type target: int
            :rtype: int
            """
            nums.sort()
            res = 0
            l = len(nums)
            for i in range(l - 2):
                if 3 * nums[i] > target:
                    return res
                j = i + 1
                k = l - 1
                while j < k:
                    if nums[i] + nums[j] + nums[k] < target:
                        res += k - j
                        j += 1
                    else:
                        k -= 1
            return res
