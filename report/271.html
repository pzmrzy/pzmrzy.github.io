    • Encode and Decode Strings

    Design an algorithm to encode a list of strings to a string. The encoded string is then sent over the network and is decoded back to the original list of strings.

    Report:
    we encode the a string into str(l) + '#' + s
when decode, find '#', from current position to # is the length of string, after # is the string
    Code:
      
    class Codec:

        def encode(self, strs):
            """Encodes a list of strings to a single string.

            :type strs: List[str]
            :rtype: str
            """
            res = ""
            for s in strs:
                l = len(s)
                res += str(l) + '#' + s
            return res

        def decode(self, s):
            """Decodes a single string to a list of strings.

            :type s: str
            :rtype: List[str]
            """
            res = []
            l = len(s)
            now = 0
            while now < l:
                ind = s.find('#', now)
                num = int(s[now:ind])
                ts = s[ind + 1: ind + num + 1]
                res.append(ts)
                now = ind + num + 1
            return res
