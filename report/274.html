    Given an array of citations (each citation is a non-negative integer) of a researcher, write a function to compute the researcher's h-index.A scientist has index h if h of his/her N papers have at least h citations each, and the other N âˆ’ h papers have no more than h citations each.

    Report:
    First sort the array, if citation[i] >= the remaining papers(l - i), return i
    Code:
      
    class Solution(object):
        def hIndex(self, citations):
            """
            :type citations: List[int]
            :rtype: int
            """
            citations.sort()
            l = len(citations)
            for i in range(l):
                if citations[i] >= l - i:
                    return l - i
            return 0
