    • First Bad Version

    find out the first bad by calling API.

    Report:
    Use binary search to find the first bad version. For binary, condition is l < r, l = mid + 1, r = mid - 1.
Finally, l == r, if l is a bad version, it is the result, or l + 1 is the result.
    Code:
      
# The isBadVersion API is already defined for you.
# @param version, an integer
# @return a bool
# def isBadVersion(version):

class Solution(object):
    def firstBadVersion(self, n):
        """
        :type n: int
        :rtype: int
        """
        l, r = 0, n
        while l < r:
            mid = (l + r) / 2
            if isBadVersion(mid):
                r = mid - 1
            else:
                l = mid + 1
        return l if isBadVersion(l) else l + 1
