    • Move Zeroes

    Given a array, move all 0's to the end of it while maintaining the relative order of the non-zero elements.

    Report:
    Record the last position of non-zero number + 1, copy a non-zero number to this position. Then replace all the number after it to zero.
For example:
[0, 1, 0, 3, 12]: i = 0, l(lastpos) = 0  ->
[0, 1, 0, 3, 12]: i = 1, l = 0  ->
[1, 1, 0, 3, 12]: i = 2, l = 1  ->
[1, 3, 0, 3, 12]: i = 3, l = 2  ->
[1, 3, 12, 3, 12]: i = 4, l = 3 ->
[1, 3, 12, 0, 0]
    Code:
      
    class Solution(object):
        def moveZeroes(self, nums):
            """
            :type nums: List[int]
            :rtype: void Do not return anything, modify nums in-place instead.
            """
            l = 0
            for n in nums:
                if n != 0:
                    nums[l] = n
                    l += 1
            for i in range(l, len(nums)):
                nums[i] = 0
