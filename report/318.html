    • Maximum Product of Word Lengths

    Given a string array words, find the maximum value of length(word[i]) * length(word[j]) where the two words do not share common letters. You may assume that each word will contain only lower case letters. If no such two words exist, return 0.

    Report:
    covert the word to bits by 1 << ord(c) - 97, store words in a dictionary, key is bits, value is the length
for two bits, bits1 and bits2, if bits1 & bits2 is zero, means they don't share same letter. then update the result.
    Code:
      
from collections import defaultdict
    class Solution(object):
        def maxProduct(self, words):
            """
            :type words: List[str]
            :rtype: int
            """
            dic = defaultdict(int)
            for w in words:
                mask = 0
                for c in w:
                    mask |= 1 << (ord(c) - 97)
                dic[mask] = max(dic[mask], len(w))
            return max([dic[x] * dic[y] for x in dic for y in dic if not x & y] + [0])
