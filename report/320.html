    • Generalized Abbreviation

    Write a function to generate the generalized abbreviations of a word.

    Report:
    dfs, record position, current string and count, two children:
move position, count + 1
move position, put count to string, put current letter to string, count = 0
    Code:
      
    class Solution(object):
        def generateAbbreviations(self, word):
            """
            :type word: str
            :rtype: List[str]
            """
            res = []
            q = [(0, "", 0)]
            l = len(word)
            while len(q) > 0:
                pos, tmp, count = q.pop()
                if l == pos:
                    res.append(tmp + (str(count) if count > 0 else ""))
                else:
                    q.append((pos + 1, tmp, count + 1))
                    q.append((pos + 1, tmp + (str(count) if count > 0 else "") + word[pos], 0))
            return res
