    Given an integer, write a function to determine if it is a power of three.

    Report:
    mod until cannot divide by 3, check whether it is 1.
    Code:
      
    class Solution(object):
        def isPowerOfThree(self, n):
            """
            :type n: int
            :rtype: bool
            """
            if n == 0:
                return False
            while n % 3 == 0:
                n /= 3
            return n == 1
