    • Longest Substring with At Most K Distinct Characters

    Given a string, find the length of the longest substring T that contains at most k distinct characters.

    Report:
    Always put the character c and its location i in the dictionary d.
If the sliding window contains less than or equal to k distinct characters, update result
Otherwise, we need to remove a character from the sliding window.Because the values in d represents the rightmost location of each character in the sliding window, in order to find the longest substring T, we need to locate the smallest location, and remove it from the dictionary, and then record the return value.
    Code:
      
    class Solution(object):
        def lengthOfLongestSubstringKDistinct(self, s, k):
            """
            :type s: str
            :type k: int
            :rtype: int
            """
            dic = {}
            res = low = 0
            for i, c in enumerate(s):
                dic[c] = i
                if len(dic) > k:
                    low = min(dic.values())
                    del dic[s[low]]
                    low += 1
                res = max(res, i - low + 1)
            return res
