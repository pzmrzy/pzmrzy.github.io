    • Moving Average from Data Stream

    Given a stream of integers and a window size, calculate the moving average of all integers in the sliding window.

    Report:
    store the numbers and their sum, if a new number in, then length large than size, then pop the left one, and update sum.
    Code:
      
    class MovingAverage(object):

        def __init__(self, size):
            """
            Initialize your data structure here.
            :type size: int
            """
            self.size = size
            self.window = []
            self.sum = 0

        def next(self, val):
            """
            :type val: int
            :rtype: float
            """
            self.sum += val
            self.window.append(val)
            l = len(self.window)
            if l > self.size:
                self.sum -= self.window.pop(0)
            return self.sum / float(min(l, self.size))
