    • UTF-8 Validation

    Given an array of integers representing the data, return whether it is a valid utf-8 encoding.

    Report:
    Use bit operation to check whether a number if valid. First decide the length of utf8 by the first number, the number should be zero after check all the nubmer.
    Code:
      
    class Solution(object):
        def validUtf8(self, data):
            """
            :type data: List[int]
            :rtype: bool
            """
            if len(data) == 0:
                return False
            ct = 0
            for n in data:
                if ct == 0:
                    if n >> 5 == 0b110:
                        ct = 1
                    elif n >> 4 == 0b1110:
                        ct = 2
                    elif n >> 3 == 0b11110:
                        ct = 3
                    elif n >> 7 != 0:
                        return False
                else:
                    if n >> 6 != 0b10:
                        return False
                    ct -= 1
            return ct == 0
