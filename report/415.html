    Given two non-negative integers num1 and num2 represented as string, return the sum of num1 and num2.

    Report:
    use a pointers for each number, start from end, and record a carry. if there still number, add it to carry, the result of this bit is carry % 10. then carry /= 10
    Code:
      
    class Solution(object):
        def addStrings(self, num1, num2):
            """
            :type num1: str
            :type num2: str
            :rtype: str
            """
            res = []
            c = 0
            l1 = len(num1)
            l2 = len(num2)
            while l1 or l2 or c:
                if l1:
                    l1 -= 1
                    c += int(num1[l1])
                if l2:
                    l2 -= 1
                    c += int(num2[l2])
                res.append(str(c % 10))
                c /= 10
            return ''.join(reversed(res))
