    • Sentence Screen Fitting

    Given a rows x cols screen and a sentence represented by a list of non-empty words, find how many times the given sentence can be fitted on the screen.

    Report:
    first join all the words together by space(include last one). st record the total length of all sentences(repeat many times).
each time st + total length of one sentence, if st % len is space, start + 1, else, start - 1 until it is space.
the result is total length / one sentence length
    Code:
      
    class Solution(object):
        def wordsTyping(self, sentence, rows, cols):
            """
            :type sentence: List[str]
            :type rows: int
            :type cols: int
            :rtype: int
            """
            s = ' '.join(sentence) + ' '
            st, l = 0, len(s)
            for i in range(rows):
                st += cols
                if s[st % l] == ' ':
                    st += 1
                else:
                    while st > 0 and s[(st - 1) % l] != ' ':
                        st -= 1
            return st / l
