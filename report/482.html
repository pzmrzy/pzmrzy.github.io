    • License Key Formatting

    representing a license key to format, upper, use - to split for each K char

    Report:
    First remove dash and covert to uppercase. Get the remainder, and these chars in a group, others each K chars in a group.
    Code:
      
    class Solution(object):
        def licenseKeyFormatting(self, S, K):
            """
            :type S: str
            :type K: int
            :rtype: str
            """
            S = S.replace('-', '').upper()
            l = len(S)
            if l < K:
                return S
            r = l % K
            res = [] if r == 0 else [S[:r]]
            for i in range(l / K):
                tmp = r + i * K
                res.append(S[tmp: tmp + K])
            return '-'.join(res)
