    • Longest Word in Dictionary through Deleting

    Given a string and a string dictionary, find the longest string in the dictionary that can be formed by deleting some characters of the given string. If there are more than one possible results, return the longest word with the smallest lexicographical order. If there is no possible result, return the empty string.

    Report:
    sort the words by length and lexicographical, for each word, check whether it can be formed by s.
Use a pointer for word, if a char in s match it, pointer move next. If the pointer can get the end of word, return this word.
    Code:
      
    class Solution(object):
        def findLongestWord(self, s, d):
            """
            :type s: str
            :type d: List[str]
            :rtype: str
            """
            d.sort(key = lambda x: (-len(x), x))
            for word in d:
                i = 0
                for c in s:
                    if i < len(word) and word[i] == c:
                        i += 1
                if i == len(word):
                    return word
            return ""
