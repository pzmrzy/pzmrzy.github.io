    • Output Contest Matches

    Now, you're given n teams, you need to output their final contest matches in the form of a string.

    Report:
    Do combination for log(2, n) time.
Each time combine the first and last, second and last two,
    Code:
      
    class Solution(object):
        def findContestMatch(self, n):
            """
            :type n: int
            :rtype: str
            """
            res = map(str, range(1, n + 1))
            while n > 1:
                res = ["(" + res[i] + ',' + res[n - i - 1] + ")" for  i in range(n >> 1)]
                n >>= 1
            return res[0]
