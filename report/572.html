    • Subtree of Another Tree

    Given two non-empty binary trees s and t, check whether tree t has exactly the same structure and node values with a subtree of s. A subtree of s is a tree consists of a node in s and all of this node's descendants. The tree s could also be considered as a subtree of itself.

Example 1:
Given tree s:

     3
    / \
   4   5
  / \
 1   2
Given tree t:
   4
  / \
 1   2
Return true, because t has the same structure and node values with a subtree of s.
Example 2:
Given tree s:

     3
    / \
   4   5
  / \
 1   2
    /
   0
Given tree t:
   4
  / \
 1   2
Return false.
    Report:
    A function to check whether two trees are same. first compare value, them compare children
for tree s and t, if s is t, or s is subtree of t, or t is subtree of s, return true
    Code:
      
    # Definition for a binary tree node.
    # class TreeNode(object):
    #     def __init__(self, x):
    #         self.val = x
    #         self.left = None
    #         self.right = None

    class Solution(object):
        def isMatch(self, s, t):
            if not(s and t):
                return s is t
            return s.val == t.val and self.isMatch(s.left, t.left) and self.isMatch(s.right, t.right)

        def isSubtree(self, s, t):
            """
            :type s: TreeNode
            :type t: TreeNode
            :rtype: bool
            """
            if self.isMatch(s, t):
                return True
            if not s:
                return False
            return self.isSubtree(s.left, t) or self.isSubtree(s.right, t)
