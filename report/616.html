    • Add Bold Tag in String

    Given a string s and a list of strings dict, you need to add a closed pair of bold tag and to wrap the substrings in s that exist in dict. If two such substrings overlap, you need to wrap them together by only one pair of closed bold tag. Also, if two substrings wrapped by bold tags are consecutive, you need to combine them.

    Report:
    we maintance a flag array, if a char in the word, label it to true
for each word in dictionary, scan the string, if match, then lavel the substring to true
finaly, base on the flag array, form the result.
    Code:
      
    class Solution(object):
        def addBoldTag(self, s, dict):
            """
            :type s: str
            :type dict: List[str]
            :rtype: str
            """
            l = len(s)
            f = [False] * len(s)
            for word in dict:
                for i in range(l - len(word) + 1):
                    if word == s[i: i + len(word)]:
                        for j in range(i, i + len(word)):
                            f[j] = True
            res = ""
            i = 0
            while i < l:
                if f[i]:
                    res += ""
                    while i < l and f[i]:
                        res += s[i]
                        i += 1
                    res += ""
                else:
                    res += s[i]
                    i += 1
            return res
