    • Maximum Average Subarray I

    Given an array consisting of n integers, find the contiguous subarray of given length k that has the maximum average value. And you need to output the maximum average value.

    Report:
    find max average equal to find max sum, use init with sum of first k number and 2 pointers, each time minus left one, add right one, and update result.
answer is the average of max sum.
    Code:
      
    class Solution(object):
        def findMaxAverage(self, nums, k):
            """
            :type nums: List[int]
            :type k: int
            :rtype: float
            """
            now = sum(nums[:k])
            res = now
            i, j = 0, k
            while j < len(nums):
                now = now - nums[i] + nums[j]
                if now > res:
                    res = now
                i += 1
                j += 1
            return float(res) / k
