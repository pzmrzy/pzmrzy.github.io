    • Split Array into Consecutive Subsequences

    You are given an integer array sorted in ascending order (may contain duplicates), you need to split them into several subsequences, where each subsequences consist of at least 3 consecutive integers. Return whether you can make such a split.

    Report:
    greedy, use a dictionary to record, the lengths of subarray, end with key. alwasy append the number to the shortest subarray.
for example [1,2,3,3,4,4,5,5]
1 -> 1
2 -> 2
3 -> 3
3 -> 1, 3
3 -> 3, 4 -> 2
4 -> 2, 4
4 -> 4
5 -> 3
5 -> 3, 5
if exsit a subarray, its length less than 3, return false
    Code:
      
    class Solution(object):
        def isPossible(self, nums):
            """
            :type nums: List[int]
            :rtype: bool
            """
            dic = {}
            for n in nums:
                if n - 1 not in dic:
                    if n not in dic:
                        dic[n] = [1]
                    else:
                        heapq.heappush(dic[n], 1)
                else:
                    l = heapq.heappop(dic[n - 1]) + 1
                    if len(dic[n - 1]) == 0:
                        del dic[n - 1]
                    if n not in dic:
                        dic[n] = []
                    heapq.heappush(dic[n], l)
            for key, arr in dic.items():
                if len(arr) > 0 and min(arr) < 3:
                    return False
            return True
