    You have 4 cards each containing a number from 1 to 9. You need to judge whether they could operated through *, /, +, -, (, ) to get the value of 24.

    Report:
    Because there are very few possible permutation of numbers and operations, so we can just enmuerate all of them.
We do it recursive, get two of numbers and combine them, then get 24 with 3 numbers, then 2 numbers, then result.
reminder: divide by zero, and the result should be float, when check the result, cannot easuly do equal.
    Code:
      
class Solution:
    def judgePoint24(self, nums):
        """
        :type nums: List[int]
        :rtype: bool
        """
        if len(nums) == 1:
            return math.isclose(nums[0], 24)
        return any(self.judgePoint24([x] + rest)
                   for a, b, *rest in itertools.permutations(nums)
                  for x in [a + b, a - b, a * b, b and a / b])
