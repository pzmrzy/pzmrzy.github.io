    Compute and return the square root of x.

    Report:
    1 Use binary search to find the soluthon of sqrt(x)
2 Use Newton method to solve the equation of x^2 - n = 0
First we assume x0 = x / 2 + 1
when xk * xk < res, xk is the result.

    x_{k} = x_{k-1} - \frac{f(x_{k-1})}{\dot{f_{k-1}}} \\
x_{k} = x_{k-1} - \frac{x_{k-1}^{2} - n}{2X_{k-1}} \\
x_{k} = (x_{k - 1} + \frac{n}{x_{k-1}})/2
    Code:
      
    class Solution(object):
        def mySqrt(self, x):
            """
            :type x: int
            :rtype: int
            """
            res = x / 2 + 1
            while res * res > x:
                res = (res + x / res) / 2
            return res
