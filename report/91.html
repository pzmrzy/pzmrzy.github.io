    A message containing letters from A-Z is being encoded to numbers using the following mapping:A->1, ...,Z->26 Given an encoded message containing digits, determine the total number of ways to decode it.

    Report:
    Use dp, assume l = len(s), dp[i] is the result of s[i:], dp[l] = 0, d[l] = 1, if s[l-1] not 0, dp[0] is the answer
dp[i] = 0                     s[i] = '0'
        d[i + 1] + dp[i + 2]  s[i: i+2] <= 26, this char can as it self or combine with the char next to it
        dp[i + 1]             else
conner case s = "", "00" in s
    Code:
      
class Solution(object):
    def numDecodings(self, s):
        """
        :type s: str
        :rtype: int
        """
        if len(s) == 0:
            return 0
        l = len(s)
        dp = [0] * (l + 1)
        dp[-1] = 1
        dp[-2] = 0 if s[-1] == '0' else 1
        for i in range(l - 2, -1, -1):
            if s[i] == '0':
                continue
            dp[i] = dp[i + 1] + dp[i + 2] if int(s[i: i+2]) <= 26 else dp[i + 1]
        return dp[0]
